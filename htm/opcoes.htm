Configurações da Extensão

A URL fornecida deve obedecer às regras de CORS (Cross-Origin Resource Sharing). Isso significa que o servidor deve permitir explicitamente o acesso de origens diferentes, configurando os cabeçalhos HTTP adequados. Para liberar o acesso, adicione os seguintes cabeçalhos no servidor:


Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: GET, POST, PUT, DELETE
Access-Control-Allow-Headers: Content-Type
